# Python Calculator

A simple desktop calculator application developed using **Python** and **Tkinter**. This project provides a graphical user interface (GUI) to perform basic arithmetic calculations efficiently.

## Features

- User-friendly graphical interface
- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Modulus (%)
- Decimal number support
- Clear button to reset input
- Error handling for invalid expressions

## Technologies Used

- Python 3
- Tkinter

## Project Structure

```
calculator.py
README.md
```

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python calculator.py
```
